# House of Models DG Landing Page

Адаптивный лендинг для студии House of Models DG на HTML, CSS и JavaScript.

## Live demo

Сайт опубликован и доступен по ссылке:

```text
https://modelsdg.ru/
```

## О проекте

Проект представляет собой лендинг для студии с информацией о компании, требованиях, контактах и формой заявки.

На сайте реализованы:

- фиксированная шапка;
- мобильное меню;
- затемнение фона при открытии меню;
- hero-блок с CTA-кнопкой;
- секция «О нас»;
- секция «Требования»;
- секция «Где мы»;
- FAQ-блок;
- форма заявки;
- загрузка фотографии в форме;
- legal-страницы: политика конфиденциальности и пользовательское соглашение;
- адаптивная вёрстка под мобильные устройства.

## Особенности реализации

- Адаптивная шапка с изменением состояния при скролле.
- Мобильное меню с overlay-затемнением.
- Плавная прокрутка к секциям по якорям.
- FAQ-аккордеон.
- Форма обратной связи.
- Загрузка и предпросмотр изображения перед отправкой.
- Отдельные страницы для политики конфиденциальности и пользовательского соглашения.
- Подготовка проекта к публикации в портфолио.

## Примечание

В репозитории размещена очищенная версия исходного кода.

Медиафайлы не включены в публичный репозиторий, так как полноценная визуальная версия сайта доступна по live-ссылке.

Файл с реальными backend-настройками не публикуется. Вместо него добавлен безопасный пример:

```text
backend/conf.example.php
```

## Безопасность

В публичный репозиторий нельзя загружать:

- реальные SMTP-логины;
- реальные SMTP-пароли;
- приватные backend-конфиги;
- файлы с доступами;
- `.env`;
- приватные изображения;
- логи;
- пользовательские заявки.
